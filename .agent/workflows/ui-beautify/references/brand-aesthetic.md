# Deep Ocean 2025 品牌审美指南

本文档详细定义音乐播放器的视觉设计语言和品牌美学标准。

---

## 设计哲学

### 核心理念

**"沉浸于音乐，专注于内容"**

界面应该像深海一样：

- **深邃** — 深色背景减少视觉干扰
- **平静** — 柔和过渡，无突兀元素
- **专业** — 精致细节，高端质感

### 设计原则

1. **内容优先**: 音乐信息是主角，UI 是配角
2. **层次分明**: 通过颜色深浅区分信息层级
3. **状态清晰**: 交互反馈即时、明确
4. **一致性**: 全局遵循统一的设计语言

## 详细色彩系统

### 主色调 Teal (青色)

选择 Teal 作为品牌主色的原因：

- 与深色背景形成鲜明对比
- 传达科技感与专业性
- 在音乐应用中独特而不俗套

## 排版系统

### 字体选择

**主字体栈**:

```
"Segoe UI Variable", "Segoe UI", "Microsoft YaHei", sans-serif
```

选择理由：

- Windows 原生支持，无需额外安装
- Variable 版本支持多种字重
- Microsoft YaHei 确保中文显示良好

### 字体规模

使用 Major Second (1.125) 比例：

| 级别 | 大小 | 行高 | 字重 | CSS |
|------|------|------|------|-----|
| Display | 28px | 36px | Bold (700) | `font-size: 28px; font-weight: 700` |
| H1 | 24px | 32px | SemiBold (600) | `font-size: 24px; font-weight: 600` |
| H2 | 18px | 26px | SemiBold (600) | `font-size: 18px; font-weight: 600` |
| H3 | 16px | 24px | Medium (500) | `font-size: 16px; font-weight: 500` |
| Body | 14px | 22px | Regular (400) | `font-size: 14px` |
| Caption | 12px | 18px | Regular (400) | `font-size: 12px` |
| Overline | 11px | 16px | Bold (700) | `font-size: 11px; font-weight: 700; text-transform: uppercase; letter-spacing: 1px` |

## 阴影与深度

本主题采用极简阴影策略，通过色彩层次而非阴影表达深度：

| 层级 | 背景色 | 说明 |
|------|--------|------|
| Level 0 | #0E1116 | 基础背景 |
| Level 1 | #111620 | 侧边栏、底栏 |
| Level 2 | #151B26 | 菜单、弹出层 |
| Level 3 | #1B2230 | 提示框 |

边框用于分隔，而非阴影：

```css
border: 1px solid #1E2633;
```

## 动画与过渡

### 时长规范

| 类型 | 时长 | 缓动函数 |
|------|------|---------|
| 快速反馈 | 100ms | ease-out |
| 状态切换 | 150ms | ease-in-out |
| 内容展开 | 200ms | ease-out |
| 页面过渡 | 300ms | ease-in-out |

### QSS 过渡 (有限支持)

PyQt6 QSS 对动画支持有限，主要通过：

- 颜色渐变效果需借助 QPropertyAnimation
- Hover 状态切换使用 CSS transition

## 图标风格

### 设计规范

- **线性图标**: 2px 线宽
- **尺寸**: 24x24px (标准)、16x16px (紧凑)、32x32px (强调)
- **颜色**: 使用 `currentColor` 继承父元素颜色
- **圆角**: 2px (与品牌圆角保持一致)

### 图标颜色

| 状态 | 颜色 |
|------|------|
| 默认 | #9AA2AF (Text-Secondary) |
| 悬停 | #E6E8EC (Text-Primary) |
| 激活 | #3FB7A6 (Teal-500) |
| 禁用 | #5E6775 (Text-Disabled) |

## 可访问性

### 对比度要求

遵循 WCAG 2.1 AA 标准：

- 正文文本对比度 ≥ 4.5:1
- 大文本 (≥18px) 对比度 ≥ 3:1
- 非文本元素对比度 ≥ 3:1

### 验证结果

| 组合 | 对比度 | 评级 |
|------|--------|------|
| #E6E8EC 在 #0E1116 上 | 14.3:1 | ✅ AAA |
| #9AA2AF 在 #0E1116 上 | 7.2:1 | ✅ AAA |
| #3FB7A6 在 #0E1116 上 | 8.1:1 | ✅ AAA |
| #5E6775 在 #0E1116 上 | 4.0:1 | ✅ AA (Large) |

### 焦点指示器

```css
    border: 2px solid #3FB7A6;
    outline: none;
}
```

## 最佳实践

### Do ✅

- 使用品牌色强调主要操作
- 保持背景层次清晰
- 为所有交互元素提供视觉反馈
- 使用适当的间距创造呼吸感

### Don't ❌

- 不要在深色背景上使用浅色边框
- 不要使用纯黑 (#000000)
- 不要忽略禁用状态的样式
- 不要使用超过 3 种不同的字号
